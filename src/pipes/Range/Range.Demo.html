Range

Range is a custom pipe to dynamically create an array of numbers based on a given range

*ngFor="let number of numbers | range : {{startNumber}} : {{endNumber}} : {{stepNumber}}"

Output: {{[] | range : startNumber : endNumber : stepNumber | json}}

Import


import {{'{'}}FuiRangePipeModule{{'}'}} from 'fuel-ui'; //module
import {{'{'}}RangePipe{{'}'}} from 'fuel-ui'; //class

Getting Started

Range is a pipe that simply takes in 3 arguments, a start number, end number, and step number.

Usage


*ngFor="let number of numbers | range : start : end : step"



export class RangeExample {{'{'}}
    numbers:number[] = [];
    start:number = 0;
    end:number = 5;
    step:number = 1;
{{'}'}}
